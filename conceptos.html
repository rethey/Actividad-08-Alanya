Conceptos de Lenguajes de Programación

Python
Python es un lenguaje de programación interpretado, de alto nivel, de propósito general y de tipado dinámico. Es muy popular por su sintaxis clara y legible, ideal para desarrollo web, análisis de datos, inteligencia artificial y automatización.
JavaScript
JavaScript es un lenguaje de programación interpretado o compilado JIT que cumple con el estándar ECMAScript. Es conocido principalmente por ser el lenguaje de programación de la web, permitiendo interactividad en los navegadores web.
Java
Java es un lenguaje de programación de propósito general, orientado a objetos, diseñado para tener la menor cantidad posible de dependencias de implementación. Es ampliamente utilizado para aplicaciones empresariales a gran escala, aplicaciones móviles Android y sistemas distribuidos.
C#
C# (pronunciado "C sharp") es un lenguaje de programación multiparadigma desarrollado por Microsoft como parte de su plataforma .NET. Combina características de lenguajes como C++ y Java, siendo muy popular para el desarrollo de aplicaciones de escritorio Windows, juegos (Unity) y desarrollo web (ASP.NET).
